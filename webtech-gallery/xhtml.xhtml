XHTML Basics

XHTML (Extensible Hypertext Markup Language) is a stricter, XML-based version of HTML. It enforces clean, well-formed markup that is consistent and compatible with modern web standards.

Key Features of XHTML:

  • All tags must be properly closed.
  • All tag names must be in lowercase.
  • Attribute values must be quoted.
  • The document must begin with a proper DOCTYPE declaration.

Example XHTML Code:

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

<html xmlns="http://www.w3.org/1999/xhtml">
  <head>
    <title>Sample Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is valid XHTML.</p>
  </body>
</html>

Notice how every tag is closed, the tag names are in lowercase, and attribute values are properly quoted.
