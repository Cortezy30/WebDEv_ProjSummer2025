Styling with CSS

CSS (Cascading Style Sheets) is used to control the look and feel of a website. It separates content from presentation, allowing for cleaner, more maintainable code.

Basic Styling Example

This paragraph is plain and has minimal styling.

This paragraph is styled using CSS. Notice the custom font, padding, background, and border effects.

CSS Syntax Example:

p {
    color: blue;
    font-size: 16px;
    margin: 10px;
}

Selectors (e.g., p) target elements, and declarations apply styles using property: value; pairs.
