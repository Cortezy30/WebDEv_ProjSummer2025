# WebDEv_ProjSummer2025
A website that I developed for my Web Development class that I took in the summer of 2025 going into my senior year of college. I had to include a bit of everything that we learned in the course, so, why not a simple gallery demonstrating each topic with its own page. Boring but was meant to showcase what I learned. The server software my teacher asked us to use was Apache and I downloaded XAMPP control panel to test, here's a link if you want to do the same https://www.apachefriends.org/ 

How to set up the project:
1.Install and Run XAMPP
Download XAMPP (https://www.apachefriends.org)
Install and start Apache server.

2.Place Website Files
Put your website folder (e.g., webtech-gallery) inside the htdocs directory of XAMPP.
Example path: C:\xampp\htdocs\webtech-gallery

3.Verify Folder Structure
Confirm files like index.xhtml, xhtml.xhtml, css.xhtml, javascript.xhtml, php.php, sessions.php, filedb.php, xml.xhtml, and data.xml are correctly located inside the folder.
(I didn't put the files in separate folders but if you'd prefer to do that for organization purposes feel free.)

4.Access Site in Browser
Navigate to http://localhost/webtech-gallery/
The server should automatically load index.xhtml.
(sidenote the url name should match the folder so if you don't name it webtech-gallery, just make sure the name in the url is the folder name you set.)
